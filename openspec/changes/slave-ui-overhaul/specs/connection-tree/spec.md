## 新增需求

### 需求:树形导航展示连接层级

系统必须在左侧面板以树形结构展示所有从站连接、从站设备和寄存器分组的层级关系。树形导航的宽度固定为 240px。

#### 场景:展示连接节点
- **当** 用户创建了一个端口为 5020 的从站连接
- **那么** 树形导航中必须显示一个连接节点，标签为 "0.0.0.0:5020"，并显示连接状态图标（绿色=运行中，红色=已停止）

#### 场景:展示从站节点
- **当** 连接下存在 slave_id=1 的从站设备
- **那么** 连接节点下必须显示一个从站子节点，标签为 "从站 1"

#### 场景:展示寄存器分组节点
- **当** 用户展开从站节点
- **那么** 必须显示四个寄存器分组子节点：0x Coils、1x Discrete Inputs、3x Input Registers、4x Holding Registers

### 需求:树节点点击切换寄存器表格内容

用户点击树形导航中的节点时，中间寄存器表格必须相应更新展示内容。

#### 场景:点击寄存器分组节点
- **当** 用户点击 "4x Holding Registers" 节点
- **那么** 中间寄存器表格必须只显示该从站的 Holding Register 类型寄存器

#### 场景:点击从站节点
- **当** 用户点击从站节点
- **那么** 中间寄存器表格必须显示该从站所有类型的寄存器

#### 场景:点击连接节点
- **当** 用户点击连接节点
- **那么** 该连接节点在树中高亮选中，工具栏的启动/停止按钮状态必须随之更新

### 需求:树节点右键菜单

用户右键点击树节点时必须弹出上下文菜单，提供对应层级的操作。

#### 场景:右键连接节点
- **当** 用户右键点击连接节点
- **那么** 必须显示上下文菜单，包含"启动连接"/"停止连接"（根据当前状态显示其一）和"删除连接"选项

#### 场景:右键从站节点
- **当** 用户右键点击从站节点
- **那么** 必须显示上下文菜单，包含"删除从站"选项

#### 场景:菜单外点击关闭
- **当** 上下文菜单已显示，用户点击菜单外任意区域
- **那么** 上下文菜单必须关闭

### 需求:树节点展开折叠

树形导航中的连接节点和从站节点必须支持展开和折叠。

#### 场景:默认展开状态
- **当** 新创建一个连接
- **那么** 该连接节点及其下的从站节点必须默认展开

#### 场景:点击箭头折叠
- **当** 用户点击节点前的展开箭头
- **那么** 该节点的子节点必须折叠隐藏，箭头方向改变
