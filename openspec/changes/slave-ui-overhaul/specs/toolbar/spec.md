## 新增需求

### 需求:顶部工具栏布局

系统必须在窗口顶部显示一个工具栏，包含所有主要操作按钮。工具栏高度固定，横跨整个窗口宽度。

#### 场景:工具栏按钮列表
- **当** 应用启动
- **那么** 工具栏必须从左到右依次显示以下按钮：新建连接、新建从站、启动连接、停止连接、关闭连接、工具

### 需求:新建连接操作

用户点击"新建连接"按钮时必须弹出端口输入对话框。

#### 场景:创建TCP连接
- **当** 用户点击"新建连接"按钮
- **那么** 必须弹出对话框，包含端口号输入框（默认值 5020）和确认/取消按钮
- **当** 用户输入端口号并点击确认
- **那么** 系统必须创建从站连接（含默认从站和寄存器），新连接出现在左侧树形导航中

### 需求:新建从站操作

用户点击"新建从站"按钮时必须在当前选中的连接下添加新的从站设备。

#### 场景:添加从站到选中连接
- **当** 用户已在树中选中一个连接节点，点击"新建从站"按钮
- **那么** 必须弹出对话框，包含从站ID输入框和确认/取消按钮
- **当** 用户输入从站ID并确认
- **那么** 系统必须在该连接下创建新从站（含默认 FC1/2/3/4 地址 0~100 寄存器），树中出现新从站节点

#### 场景:未选中连接时新建从站
- **当** 用户未选中任何连接节点，点击"新建从站"按钮
- **那么** 按钮必须处于禁用状态，不响应点击

### 需求:启动停止连接操作

工具栏的启动/停止按钮必须根据当前选中连接的状态动态启用或禁用。

#### 场景:选中已停止的连接
- **当** 用户在树中选中一个状态为 Stopped 的连接
- **那么** "启动连接"按钮必须启用，"停止连接"按钮必须禁用

#### 场景:选中运行中的连接
- **当** 用户在树中选中一个状态为 Running 的连接
- **那么** "启动连接"按钮必须禁用，"停止连接"按钮必须启用

#### 场景:未选中任何连接
- **当** 树中没有选中任何连接节点
- **那么** "启动连接"和"停止连接"按钮必须同时禁用

### 需求:工具菜单

工具栏的"工具"按钮必须提供 CRC/LRC 计算、地址转换等辅助工具入口。

#### 场景:打开工具弹窗
- **当** 用户点击"工具"按钮
- **那么** 必须弹出工具面板或下拉菜单，包含 CRC16 计算、LRC 计算、PLC 地址转换等功能入口
