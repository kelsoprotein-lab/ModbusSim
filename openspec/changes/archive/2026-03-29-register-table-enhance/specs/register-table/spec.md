## 修改需求

### 需求:表格滚动

寄存器表格在内容超出可视区域时必须可以垂直滚动，表头必须保持固定在顶部。

#### 场景:寄存器数量超出视口
- **当** 从站包含 101 个 Holding Register，表格区域高度不足以显示全部行
- **那么** 表格必须出现垂直滚动条，用户可以滚动查看所有寄存器
- **那么** 表头（地址、名称、值、备注）必须固定在滚动区域顶部

#### 场景:鼠标滚轮滚动
- **当** 用户在表格区域使用鼠标滚轮
- **那么** 表格内容必须跟随滚动

### 需求:键盘导航

表格必须支持键盘上下箭头在寄存器行之间移动选中状态。

#### 场景:按下箭头键移动选中
- **当** 用户已选中某一行，按下 ↓ 键
- **那么** 选中状态必须移动到下一行，如果下一行不在可视区域则必须自动滚动

#### 场景:按上箭头键移动选中
- **当** 用户已选中某一行，按下 ↑ 键
- **那么** 选中状态必须移动到上一行，如果上一行不在可视区域则必须自动滚动

#### 场景:在第一行按上箭头
- **当** 用户已选中第一行，按下 ↑ 键
- **那么** 选中状态必须保持在第一行不变

#### 场景:在最后一行按下箭头
- **当** 用户已选中最后一行，按下 ↓ 键
- **那么** 选中状态必须保持在最后一行不变
