# v0.14.1 — 2026-05-01

> Tauri slave 真正用上 core 变异 API + 测试钉死四类寄存器变异
>
> Tauri slave finally calls the core mutation API; six unit tests pin behaviour for all four register types.

## 亮点 / Highlights

- ♻️ **Tauri slave 复用 core 变异 API** — `random_mutate_registers` 由 75 行就地实现改为单行 `apply_random_mutation_thread(&types)`,与 egui slave 行为完全一致。
  *Tauri slave's mutation command shrinks from 75 lines to a single core call.*
- 🧪 **变异行为单元测试覆盖** — 新增 `tests/random_mutation.rs` 6 用例,workspace 测试达 **280 / 0 失败**(此前 274 + 本版本 6)。
  *Six new unit tests pin all four register-type mutations; workspace at **280 passing, 0 failing**.*
- 🩺 **后端 + 前端诊断日志** — Tauri 侧新增 per-type addr count + 实际变异数日志,排查"变异请求来了但 UI 没刷"类问题不再靠猜。
  *Both backend (`log::debug!`) and frontend (`console.debug`) record mutation diagnostics.*
- 📜 **项目级 CLAUDE.md 入库** — `.claude/CLAUDE.md` 四条 LLM 协作准则随仓库共享。
  *Project-level `.claude/CLAUDE.md` is now version-controlled.*
